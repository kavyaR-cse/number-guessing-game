# Number Guessing Game (Java)

A simple console-based Java game where the player tries to guess a randomly generated number.

## How to Play

1. The program selects a random number between 1 and 100.
2. The player inputs their guesses.
3. The program gives feedback:
   - "Too low!" if your guess is smaller
   - "Too high!" if your guess is bigger
4. The game continues until the player guesses correctly.
5. The program shows how many attempts it took to guess correctly.

## Concepts Used

- Random number generation (`java.util.Random`)
- Loops (`while`)
- Conditional statements (`if-else`)
- User input (`Scanner`)
